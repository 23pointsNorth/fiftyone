FiftyOne Teams Install Guide
=============================

.. default-role:: code

The purpose of this document is to guide you through the install process for FiftyOne Teams.  This document will set expectations regarding what to expect from Voxel51, the FiftyOne Teams deployment options available to you, and to communicate recommendations from the Voxel51 team. It is designed for you to read the portions that are relevant to your specific installation situation.
